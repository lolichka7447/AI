## 2. Архитектура

Модуль расширяет Report module — менеджер может создавать отчёты за подчинённых.

### Роли доступа
| Роль | Доступ |
|------|--------|
| Project Manager | Сотрудники своих проектов |
| Department Manager / Tech Lead | Подчинённые сотрудники |
| Admin | Все сотрудники |
